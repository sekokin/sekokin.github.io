更新 Scoop 配置#

目前在 Scoop 过程中使用了 scoop-proxy-cn 作为主仓库。但 scoop-proxy-cn 提供了两种访问方式：

  1. gitee（用户登录）
  2. github（偶尔无法方案）。

本机器使用 scoop-proxy-cn 选择 github 时候，因此还需要新的一次代理。即： 网站 获取最新地址，然后执行脚本修改这个仓库

echo "删除以前配置， 其中main 为scoop update主仓库"
scoop bucket rm main

$PREFIX="https://ghfast.top/"
echo "修改scoop update 需要配置内容"
scoop config SCOOP_REPO $PREFIX'github.com/ScoopInstaller/Scoop.git'
scoop bucket add main $PREFIX'github.com/lzwme/scoop-proxy-cn'
scoop update
echo "# spc仓库，执行 scoop install 需要"
